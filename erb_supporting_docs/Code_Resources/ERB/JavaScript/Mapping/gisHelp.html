EJScreen

Usages

  1. View the raw EJ data of one county.

  1.1 Open the accordion panel labeled "EJScreen".

  1.2 Select the radio button labeled "On" to turn the EJ layer on.

  1.3 Select the radio button labeled "Single County" if not already selected.

  1.4 Select any county on the map.

  1.5 Select the radio button labeled "Raw Data".

  1.6 Select the newly populated checkbox labeled "All".

  1.7 Select the button labeled "Retrieve Results".

  2. Compare the raw EJ data of two counties.

  2.1 Open the accordion panel labeled "EJScreen".

  2.2 Select the radio button labeled "On" to turn the EJ layer on.

  2.3 Select the radio button labeled "Multi County".

  2.4 Select any two counties on the map.

   *To change an already selected county, select on the box containing the GEOID then select a new county

  2.5 Select the radio button labeled "Raw Data".

  2.6 Select the newly populated checkbox labeled "All".

  2.7 Select the button labeled "Retrieve Results".

 3. Visualize the EJ data of one county.

  3.1 Follow steps 1.1 - 1.7

  3.2 Select the menu item labeled "View" at the top of the window.

  3.3 Select the option "Visualize Data".

 4. Visualize the comparison of EJ data from two counties.

  4.1 Follow steps 2.1 - 2.7

  4.2 Select the menu item labeled "View" at the top of the window.

  4.3 Select the option "Visualize Data".

 5. View the EJ report of one county.

  5.1 Follow steps 1.1 - 1.4

  5.2 Select the radio button labeled "EJ Report".

  5.3 Select the button labeled "Retrieve Results".

 6. Compare the EJ reports of two counties.

  6.1 Follow steps 2.1 - 2.4

  6.2 Select the radio button labeled "EJ Report".

  6.3 Select the button labeled "Retreive Results".

 7. Link a dst file to one county and view the data on the map.

  7.1 Open the accordion panel labeled "EJScreen".

  7.2 Select the radio button labeled "On" to turn the EJ layer on.

  7.3 Select the radio button labeled "Single County".

  7.4 Select the checkbox labeled "Link GEOID to DST file".

  7.5 Select any county on the map.

  7.6 Select the button labeled "Choose File", and select a dst file was output from the MSWDST.

  7.7 Select the checkbox labeled "Add DST layer to map".

  7.8 Hover over the selected county to view the data.

 8. Link two dst files to two counties to compare their data on the map.

  8.1 Open the accordion panel labeled "EJScreen"

  8.2 Select the radio button labeled "On" to turn the EJ layer on.

  8.3 Select the radio button labeled "Multi County".

  8.4 Select the checkbox labeled "Link GEOID to DST file".

  8.5 Select any two counties on the map.

  8.6 Select the button labeled "Choose File", and select a dst file that was output from the MSWDST.

   *Repeat the above step for the second county

  8.7 Select the checkbox labeled "Add DST layer to map".

  8.8 Hover over the selected counties to view the data.

Facilities

Usages

 1. View the locations of multiple types of facilites.

  1.1 Open the accordion panel labeled "Facilities".

  1.2 Select any of the facility type checkboxes.

   *Can have multiple selected at once.

 2. Access information of a selected facility.

  2.1 Open the accordion panel labeled "Facilities".

  2.2 Select any of the facility type checkboxes.

  2.3 Select any of the markers that are located on the map.
